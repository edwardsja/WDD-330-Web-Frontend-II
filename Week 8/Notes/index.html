Transform and Transitions

The order of transforms matter. If you translate right 40 pixels, and then rotate, the result will be completely different than if you rotate then translate right.

Using animation keyframes gives much more control over transitions

Canvas, SVG, and Drag and Drop

Canvas width and height creates a box, but CSS canvas width and height creates a "viewport" of the canvas.

Canvas can manipulate individual pixels of an image
