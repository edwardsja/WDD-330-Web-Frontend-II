Notes

Forms

input.addEventListener('blur', () => alert('blurred'), false);
Adds a blur to element after focus is taken away

input.addEventListener('change', () => alert('changed'), false);
Once the user changes anything in the input box, the change event is flagged

Modular

variables and functions can be created outside the main.js file and referenced by the use of import

import { PI } from './pi.js'; --variable import
import { mean, variance } from './stats.js'; -- function import
You do not have to import all functions or variables that are available
