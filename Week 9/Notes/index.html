Week 9 Notes

Window Object

window.setTimeout() will invoke the callback function once after the miliseconds are up
window.setInterval() invokes the callback function an infinite number of times after the miliseconds are up
Other code continues to run while these methods are counting down time

HTML5 API

Using < audio > or < video > tags will display audio and video elements respectively.
.play() method to play
.pause() method to pause
.volume = [NUMBER BETWEEN 0 AND 1] to set the volume
.muted method to mute
